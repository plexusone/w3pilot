## [v0.4.0] - 2026-03-21

### Highlights

- Semantic selectors for AI-native element finding by accessibility attributes
- Full storage state management with sessionStorage support
- Init scripts for pre-page JavaScript injection
- 25+ new MCP tools for dialogs, network, storage, tabs, and tracing

### Added

- Semantic selectors: find elements by `role`, `text`, `label`, `placeholder`, `testid`, `alt`, `title`, `xpath`, and `near` ([`131bef9`](https://github.com/plexusone/vibium-go/commit/131bef9))
- `Vibe.Find()` and `Vibe.FindAll()` now accept `FindOptions` for semantic filtering ([`131bef9`](https://github.com/plexusone/vibium-go/commit/131bef9))
- `Element.Find()` and `Element.FindAll()` for scoped semantic search within parent elements ([`131bef9`](https://github.com/plexusone/vibium-go/commit/131bef9))
- MCP tools (`click`, `type`, `fill`, `press`) support semantic selector parameters ([`131bef9`](https://github.com/plexusone/vibium-go/commit/131bef9))
- `Vibe.AddInitScript()` to inject JavaScript before page scripts on every navigation ([`b7bb6df`](https://github.com/plexusone/vibium-go/commit/b7bb6df))
- `add_init_script` MCP tool for runtime script injection ([`b7bb6df`](https://github.com/plexusone/vibium-go/commit/b7bb6df))
- `--init-script` CLI flag for `vibium launch` and `vibium mcp` commands ([`b7bb6df`](https://github.com/plexusone/vibium-go/commit/b7bb6df))
- `Vibe.StorageState()` now captures sessionStorage in addition to cookies and localStorage ([`d24d54c`](https://github.com/plexusone/vibium-go/commit/d24d54c))
- `Vibe.SetStorageState()` restores complete browser state including sessionStorage ([`d24d54c`](https://github.com/plexusone/vibium-go/commit/d24d54c))
- `Vibe.ClearStorage()` clears all storage types (cookies, localStorage, sessionStorage) ([`d24d54c`](https://github.com/plexusone/vibium-go/commit/d24d54c))
- `clear_storage` MCP tool to clear all browser storage ([`d24d54c`](https://github.com/plexusone/vibium-go/commit/d24d54c))
- MCP tracing tools: `start_trace`, `stop_trace`, `start_trace_chunk`, `stop_trace_chunk`, `start_trace_group`, `stop_trace_group` ([`abf0ed8`](https://github.com/plexusone/vibium-go/commit/abf0ed8))
- MCP dialog tools: `handle_dialog`, `get_dialog` ([`f2ffad9`](https://github.com/plexusone/vibium-go/commit/f2ffad9))
- MCP network tools: `get_network_requests`, `clear_network_requests`, `route`, `unroute`, `route_list`, `network_state_set` ([`f2ffad9`](https://github.com/plexusone/vibium-go/commit/f2ffad9))
- MCP localStorage tools: `localstorage_get`, `localstorage_set`, `localstorage_list`, `localstorage_delete`, `localstorage_clear` ([`f2ffad9`](https://github.com/plexusone/vibium-go/commit/f2ffad9))
- MCP sessionStorage tools: `sessionstorage_get`, `sessionstorage_set`, `sessionstorage_list`, `sessionstorage_delete`, `sessionstorage_clear` ([`f2ffad9`](https://github.com/plexusone/vibium-go/commit/f2ffad9))
- MCP tab tools: `list_tabs`, `select_tab`, `close_tab` ([`f2ffad9`](https://github.com/plexusone/vibium-go/commit/f2ffad9))
- MCP testing tools: `verify_value`, `verify_list_visible`, `generate_locator`, `fill_form` ([`f2ffad9`](https://github.com/plexusone/vibium-go/commit/f2ffad9))
