# Release Notes - v0.5.0

Project renamed from vibium-go to webpilot. Native Chrome launcher, video recording, WebSocket monitoring, and page event listeners.

## Highlights

**Major rename and native browser support.**

This release renames the project from `vibium-go` to `webpilot` with a new module path, package name, and CLI. The external clicker dependency is replaced with a native Chrome launcher that auto-installs the browser. New features include video recording, WebSocket monitoring, and page event listeners.

## Breaking Changes

### Project Rename

| Before | After |
|--------|-------|
| `github.com/plexusone/vibium-go` | `github.com/plexusone/webpilot` |
| `package vibium` | `package webpilot` |
| `*Vibe` | `*Pilot` |
| `vibium` CLI | `webpilot` CLI |
| `vibium-mcp` | `webpilot-mcp` |
| `vibium-rpa` | `webpilot-rpa` |
| `VIBIUM_DEBUG` | `WEBPILOT_DEBUG` |
| `VIBIUM_HEADLESS` | `WEBPILOT_HEADLESS` |
| `VIBIUM_SKIP_BROWSER_DOWNLOAD` | `WEBPILOT_SKIP_BROWSER_DOWNLOAD` |

### Migration Guide

```go
// Before (v0.4.x)
import vibium "github.com/plexusone/vibium-go"

vibe, err := vibium.Launch(ctx)
vibe.Go(ctx, "https://example.com")
elem, err := vibe.Find(ctx, "button", nil)

// After (v0.5.0)
import "github.com/plexusone/webpilot"

pilot, err := webpilot.Launch(ctx)
pilot.Go(ctx, "https://example.com")
elem, err := pilot.Find(ctx, "button", nil)
```

CLI migration:

```bash
# Before
vibium launch --headless
vibium mcp
VIBIUM_DEBUG=1 vibium run test.json

# After
webpilot launch --headless
webpilot mcp
WEBPILOT_DEBUG=1 webpilot run test.json
```

## New Features

### Native Chrome Launcher

The browser is now bundled and auto-installed. No need for external clicker binary:

```go
// Just works - Chrome is downloaded automatically on first use
pilot, err := webpilot.Launch(ctx)
```

To skip auto-download (if using system Chrome):

```bash
export WEBPILOT_SKIP_BROWSER_DOWNLOAD=1
```

### Video Recording

Record browser sessions as video files:

```go
// Start recording
video, err := pilot.StartVideo(ctx, &webpilot.VideoOptions{
    Dir: "./recordings",
})

// Perform actions...
pilot.Go(ctx, "https://example.com")
elem, _ := pilot.Find(ctx, "button", nil)
elem.Click(ctx, nil)

// Stop and get video path
path, err := pilot.StopVideo(ctx)
fmt.Println("Video saved to:", path)
```

### WebSocket Monitoring

Monitor WebSocket connections opened by the page:

```go
err := pilot.OnWebSocket(ctx, func(ws *webpilot.WebSocketInfo) {
    fmt.Println("WebSocket opened:", ws.URL)

    ws.OnMessage(func(msg *webpilot.WebSocketMessage) {
        fmt.Printf("WS %s: %s\n", msg.Direction, msg.Data)
    })

    ws.OnClose(func(code int, reason string) {
        fmt.Printf("WS closed: %d %s\n", code, reason)
    })
})
```

### Page Event Listeners

Subscribe to page events:

```go
// Console messages
pilot.OnConsole(ctx, func(msg *webpilot.ConsoleMessage) {
    fmt.Printf("[%s] %s\n", msg.Type, msg.Text)
})

// Page errors
pilot.OnPageError(ctx, func(err *webpilot.PageError) {
    fmt.Printf("Page error: %s\n", err.Message)
})

// Network requests
pilot.OnRequest(ctx, func(req *webpilot.Request) {
    fmt.Printf("Request: %s %s\n", req.Method, req.URL)
})

// Network responses
pilot.OnResponse(ctx, func(resp *webpilot.Response) {
    fmt.Printf("Response: %d %s\n", resp.Status, resp.URL)
})

// Dialogs
pilot.OnDialog(ctx, func(dialog *webpilot.Dialog) {
    fmt.Printf("Dialog: %s - %s\n", dialog.Type, dialog.Message)
})
```
