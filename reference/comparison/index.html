    Feature Comparison¶

    This document compares vibium-go against other Vibium clients (Java, JavaScript, Python) and the Playwright MCP server.

    Overview¶

    vibium-go is a Go port of the official Vibium clients. This comparison helps track feature parity and identify gaps.

    Client Language Repository
    vibium-go Go plexusone/vibium-go
    vibium-java Java VibiumDev/vibium/clients/java
    vibium-js JavaScript VibiumDev/vibium/clients/javascript
    vibium-py Python VibiumDev/vibium/clients/python
    playwright-mcp TypeScript microsoft/playwright-mcp
